# cnweb-worldcupwager-notes

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面，内容围绕特定主题的笔记、参考与展示。  
所有页面均为静态 HTML 文件，可直接在浏览器中打开查看，无需后端支持。

## 目录说明

```
.
├── index.html          # 仓库主页入口（可选）
├── pages/              # 归档的 HTML 页面存放目录
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── README.md           # 本文件
```

- `pages/` 目录下存放所有归档的 HTML 页面，文件名按内容或编号命名。
- 每个页面均为独立完整 HTML，可单独访问。

## 页面归档说明

- 所有页面均为静态内容，不依赖外部服务或数据库。
- 页面内容可能包含笔记、参考信息或示例展示，仅供学习与参考。
- 不针对任何特定域名或网站，不提供任何形式的营销或诱导信息。

## 维护说明

- 仓库定期更新，新增或修订页面会通过 commit 记录说明。
- 欢迎通过 Issues 提交建议或报告页面问题。
- 如需引用或转载页面内容，请保留原始来源链接（如有）并注明出处。
